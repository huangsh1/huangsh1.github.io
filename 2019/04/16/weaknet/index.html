操作步骤：

  • 进入开发者选项
  • 选择NETWORK LINK CONDITIONER
  • 将ENABLE打开
  • 选择对应的网络环境，100%Loss、3G、Very Bad Network、WiFi等
  • 自定义网络环境：Add a profile
  • 备注：bandwidth是带宽，以k/s为单位；packet loss是丢包率，按百分比计算；delay延时，单位是毫秒，1000毫秒等于1秒。
